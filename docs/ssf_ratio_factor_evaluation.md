# 社保基金持股比例因子评估

## 1. 测试口径

- 因子定义：`ssf_total_hold_ratio` = 社保基金在该股票十大流通股东中的总持股比例之和
- 数据来源：从 `top10_floatholders` 表读取最近 12 期公告数据
- 持有人识别：基于 `top10_floatholder.ssf_detector.is_social_security_holder`
- 时间对齐：使用公告日期（`ann_date`），对每个交易日向前填充最近一次公告的因子值
- 脚本：`factor/alphapurify_ssf_ratio.py`
- 运行参数：`--start-date 2024-01-01 --max-stocks 100`

## 2. 评估结果

- 面板规模：`rows=8090`, `symbols=20`, `dates=573`
- 持有期设置：`1个月=20日，1季度=60日，半年=120日，1年=240日`
- Mean IC：
  - 1个月（20日）: `-0.0444`
  - 1季度（60日）: `-0.0302`
  - 半年（120日）: `-0.0920`
  - 1年（240日）: `-0.0643`

## 3. 结果解释

在当前样本下，社保基金总持股比例与未来收益呈负相关，中长期表现更明显。

这意味着：

- 社保基金持股比例越高的标的，中长期相对收益越弱；
- 社保基金持股比例越低（或无持仓）的标的，中长期相对收益越强。

该因子与社保基金个数因子表现类似，均呈现负相关特征。可能原因：

1. **样本限制**：仅 20 只股票有社保基金持股记录，样本规模较小；
2. **信息滞后**：社保基金持股数据基于季度公告，披露时市场可能已充分定价；
3. **风格轮动**：测试区间内，机构重仓股（包括社保基金持股）可能经历风格轮动压力。

建议结合持股比例变化（`ssf_total_hold_ratio_change`）等动态信号，以及扩大样本范围进行验证。
