# 波动率因子评估（20 日收益率波动率）

## 1. 测试口径

- 因子定义：`volatility = pct_change(close).rolling(20).std()`
- 脚本：`factor/alphapurify_volatility.py`
- 运行参数：`--start-date 2024-01-01 --max-stocks 50`

## 2. 评估结果

- 面板规模：`rows=27250`, `symbols=50`, `dates=545`
- Mean IC：
  - 1D: `-0.0643`
  - 5D: `-0.1007`
  - 10D: `-0.1193`

最新截面（`2026-05-08`）：

- 波动率均值：`0.0257`
- 波动率标准差：`0.0115`
- Top5（高波动）：`000070(0.0559)`, `000062(0.0478)`, `000078(0.0478)`, `000010(0.0439)`, `000056(0.0430)`
- Bottom5（低波动）：`000088(0.0076)`, `000089(0.0097)`, `000001(0.0108)`, `000028(0.0130)`, `000058(0.0132)`

## 3. 结果解释

在当前样本下，波动率因子在 1D/5D/10D 持有期的 IC 均为负，且持有期越长负相关越明显。  
这说明该口径下的“高波动”更接近风险惩罚信号，而不是收益增强信号：

- 高波动标的后续相对更弱；
- 低波动标的后续相对更强。

因此，在当前样本与参数下，更适合作为**反向使用**（低波动偏多、高波动偏空），不建议将“高波动”直接当作看多信号单独使用。
