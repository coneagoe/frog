# OBOS 因子评估与后续实现方向

## 1. 当前结论（基于现有回测口径）

`indicator/obos.py` 的 OBOS 因子在当前样本下表现为**反转信号**，而非趋势信号。

- 样本规模：`rows=21364`, `symbols=98`, `dates=218`
- Mean IC：
  - 1D: `-0.0270`
  - 5D: `-0.0659`
  - 10D: `-0.0748`

结论解释：OBOS 越高，后续收益越弱；OBOS 越低，后续收益越强。并且在 5/10 日持有期上负相关更明显。

## 2. 因子定位建议

OBOS 应定位为**短中期反转子因子**，不建议单独作为主策略驱动。

### 优点

- 逻辑可解释：超买/超卖后的均值回归
- 计算简单，易于在多标的横截面统一使用
- 在当前测试中方向稳定（IC 持续为负）

### 风险与局限

- 对市场状态敏感：趋势行情中可能明显失效
- 易受极端波动、涨跌停、流动性不足影响
- 与短期反转/乖离类因子可能存在较高重叠，增量信息有限

## 3. 后续实现建议（待落地）

1. 信号方向标准化：将 OBOS 取负后纳入统一“高分看多”框架，或在策略层按反转方向使用分位信号。
2. 交易可行性过滤：叠加基础过滤条件（流动性、ST/停牌、异常波动、涨跌停可交易性）。
3. 稳定性分层验证：按市场状态（震荡/趋势）、市值、波动率分组检验 IC 与分层收益。
4. 组合化接入：作为反转模块中的一个子信号，与动量、质量、波动等信号做相关性约束与权重分配。
5. 监控与迭代：上线后持续监控滚动 IC、分位收益与换手成本，触发阈值后动态降权/停用。

## 4. 实现验收标准（建议）

- 至少在 1D/5D/10D 三个持有期输出稳定统计；
- 在可交易过滤后，分位组合收益与 IC 方向一致；
- 提供滚动窗口稳定性报告（IC、分位差、换手/成本）；
- 与现有因子组合后，能证明有增量贡献（而非仅重复暴露）。
