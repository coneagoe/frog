# 动量因子评估（20 日收益率动量）

## 1. 测试口径

- 因子定义：`momentum = close / close.shift(20) - 1`
- 脚本：`factor/alphapurify_momentum.py`
- 运行参数：`--start-date 2024-01-01 --max-stocks 50`

## 2. 评估结果

- 面板规模：`rows=27250`, `symbols=50`, `dates=545`
- Mean IC：
  - 1D: `-0.0469`
  - 5D: `-0.0840`
  - 10D: `-0.1048`

最新截面（`2026-05-08`）：

- 动量均值：`0.0643`
- 动量标准差：`0.1694`
- Top5（高动量）：`000066(0.6204)`, `000007(0.3992)`, `000026(0.3926)`, `000062(0.3185)`, `000060(0.3097)`
- Bottom5（低动量）：`000012(-0.0820)`, `000065(-0.1011)`, `000004(-0.2561)`, `000078(-0.2622)`, `000010(-0.3333)`

## 3. 结果解释

在当前样本下，20 日动量与未来收益呈负相关，且持有期从 1D 拉长到 10D 后负相关更明显。  
这意味着该口径下的“动量”更接近**反转信号**而非趋势延续信号：

- 过去 20 日涨幅高的标的，后续相对更弱；
- 过去 20 日涨幅低（或下跌更多）的标的，后续相对更强。

因此，在当前样本与参数下，更适合作为反向使用（高动量偏空、低动量偏多），不建议直接按趋势跟随方向单独使用。
