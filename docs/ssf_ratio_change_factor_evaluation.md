# 社保基金持股比例变化因子评估

## 1. 测试口径

- 因子定义：`ssf_total_hold_ratio_change` = 社保基金总持股比例相比上次公告的变化量
- 数据来源：从 `top10_floatholders` 表读取最近 12 期公告数据
- 持有人识别：基于 `shareholder_monitor.ssf_detector.is_social_security_holder`
- 计算逻辑：对每个股票按公告日期排序，计算 `ssf_total_hold_ratio.diff()`
- 时间对齐：使用公告日期（`ann_date`），对每个交易日向前填充最近一次公告的因子值
- 脚本：`factor/alphapurify_ssf_ratio_change.py`
- 运行参数：`--start-date 2024-01-01 --max-stocks 100`

## 2. 评估结果

- 面板规模：`rows=6255`, `symbols=15`, `dates=573`
- 持有期设置：`1个月=20日，1季度=60日，半年=120日，1年=240日`
- Mean IC：
  - 1个月（20日）: `0.0141`
  - 1季度（60日）: `0.0020`
  - 半年（120日）: `-0.0856`
  - 1年（240日）: `-0.1166`

## 3. 结果解释

在当前样本下，社保基金持股比例变化与未来 1个月 收益呈**正相关**，但随着持有期拉长，相关性转为负值。

这意味着：

- 社保基金增持（持股比例上升）的标的，短期（1个月内）相对收益更强；
- 但中长期（半年及以上）表现转弱，可能存在均值回归效应。

**该因子是三个社保基金相关因子中唯一在短期呈现正相关的因子**，与传统认知部分一致（短期跟随机构增减仓）。

相比静态的持股个数和持股比例，**持股比例变化**更能反映社保基金的主动交易意图，因此具有更好的短期预测性。可能原因：

1. **动态信号**：变化量相比绝对水平，更能捕捉机构近期操作意图；
2. **短期超额收益**：社保基金增持可能反映对该股的看好，市场存在短期追随效应；
3. **长期回归**：机构增持后的标的可能在中长期面临估值压力或风格轮动，导致相关性转负；
4. **样本有效性**：虽然样本量更小（15 只股票），但这些股票均经历了社保基金持仓变化，信号更纯净。

建议优先使用该因子的短期信号（1个月持有期），并可考虑：
- 与基本面因子组合，筛选增持且基本面改善的标的；
- 与市值、行业等因子结合，控制风格暴露；
- 设置合理止盈止损，避免长期持有导致的均值回归风险；
- 扩大样本至全市场，验证因子在更大范围内的有效性。
