# 社保基金持股比例变化因子评估

## 1. 测试口径

- 因子定义：`ssf_total_hold_ratio_change` = 社保基金总持股比例相比上次公告的变化量
- 数据来源：从 `top10_floatholder` 表读取最近 12 期公告数据
- 持有人识别：基于 `shareholder_monitor.ssf_detector.is_social_security_holder`
- 计算逻辑：对每个股票按公告日期排序，计算 `ssf_total_hold_ratio.diff()`
- 时间对齐：使用公告日期（`ann_date`），对每个交易日向前填充最近一次公告的因子值
- 脚本：`factor/alphapurify_ssf_ratio_change.py`
- 运行参数：`--start-date 2024-01-01 --max-stocks 100`

## 2. 评估结果

- 面板规模：`rows=6255`, `symbols=15`, `dates=573`
- Mean IC：
  - 1D: `0.0174`
  - 5D: `0.0194`
  - 10D: `0.0244`

## 3. 结果解释

在当前样本下，社保基金持股比例变化与未来收益呈**正相关**，且持有期拉长到 10D 后正相关更明显。

这意味着：

- 社保基金增持（持股比例上升）的标的，后续相对收益更强；
- 社保基金减持（持股比例下降）的标的，后续相对收益更弱。

**该因子是三个社保基金相关因子中唯一呈现正相关的因子**，与传统认知一致（跟随机构增减仓）。

相比静态的持股个数和持股比例，**持股比例变化**更能反映社保基金的主动交易意图，因此具有更好的预测性。可能原因：

1. **动态信号**：变化量相比绝对水平，更能捕捉机构近期操作意图；
2. **超额收益来源**：社保基金增持可能反映对该股的看好，而市场对此信息的反应可能存在一定滞后；
3. **样本有效性**：虽然样本量更小（15 只股票），但这些股票均经历了社保基金持仓变化，信号更纯净。

建议优先使用该因子，并可考虑：
- 与基本面因子组合，筛选增持且基本面改善的标的；
- 与市值、行业等因子结合，控制风格暴露；
- 扩大样本至全市场，验证因子在更大范围内的有效性。
